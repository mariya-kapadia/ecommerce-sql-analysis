# E-commerce Customer & Sales Analysis (SQL Project)

## Project Overview

This project analyzes an e-commerce dataset using SQL to extract business insights related to customer behavior, sales performance, and product demand. The analysis is further visualized using Power BI to create an interactive dashboard.

## Tools Used

* SQL (SQLite)
* Power BI
* Data Analysis Techniques

## Dataset

Dataset source: Kaggle (Olist E-commerce Dataset)

The dataset includes multiple tables such as customers, orders, order items, products, and payments.

## Key SQL Analysis

* Total revenue calculation
* Top 10 customers by revenue
* Most popular products
* Monthly sales trends
* Revenue by product category
* Average order value
* Repeat vs new customers

## Advanced SQL Techniques

* JOIN operations
* GROUP BY and aggregations
* Subqueries
* Window functions (RANK, running total)

## Key Insights

* Total revenue generated is approximately 16M
* A small group of customers contributes significantly to revenue
* Certain products dominate order volume
* Sales vary monthly indicating seasonal trends
* Repeat customers play a key role in revenue generation

## Dashboard

The Power BI dashboard includes:

* Sales trend over time
* Revenue by category
* Top products
* Top customers
